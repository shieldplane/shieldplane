# Shieldplane

Shieldplane is a UI-first platform for governed incident response and remediation in Kubernetes environments.

This public repository is the GitHub home for:

- bug reports
- feature requests
- discussions
- public roadmap
- public release notes

## Start here

- Website and public docs: https://shieldplane.com
- Documentation hub: https://shieldplane.com/docs/
- Product overview: https://shieldplane.com/docs/overview.html
- Installation docs: https://shieldplane.com/docs/installation.html
- Public release notes: [RELEASES.md](RELEASES.md)
- Discussions: https://github.com/shieldplane/shieldplane/discussions

## Use this repository for

- opening public issues
- requesting features
- discussing product direction
- following public roadmap and release notes

## Support and security

- Support: see [SUPPORT.md](SUPPORT.md)
- Security reporting: see [SECURITY.md](SECURITY.md)

This repository is intentionally public-facing and community-oriented.

The commercial product core, internal automation, and private implementation details are not maintained here.
