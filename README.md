# Shieldplane

Shieldplane is a UI-first platform for governed incident response and remediation in Kubernetes environments.

This public repository is the GitHub home for:

- bug reports
- feature requests
- discussions
- public roadmap
- public release notes

## Start here

- Website and public docs: https://shieldplane.com
- Documentation hub: https://shieldplane.com/docs/
- Product overview: https://shieldplane.com/docs/overview.html
- Installation docs: https://shieldplane.com/docs/installation.html
- Public release notes: [RELEASES.md](RELEASES.md)
- Discussions: https://github.com/shieldplane/shieldplane/discussions

## Install surfaces

For evaluation and installation guidance, start with the public documentation:

- Quickstart: https://shieldplane.com/docs/quickstart.html
- Installation: https://shieldplane.com/docs/installation.html

Current distribution surfaces:

- Helm chart OCI: `oci://ghcr.io/shieldplane/charts/shieldplane`
- Versioned release bundle for authorized customers and evaluators:
  - https://github.com/shieldplane/shieldplane-core/releases/tag/v0.1.4.post4

The versioned release bundle currently includes:

- `shieldplane-bundle-0.1.4.post4.tar.gz`
- `shieldplane-bundle-0.1.4.post4.tar.gz.sha256`

Customer-specific licensing material is delivered separately during onboarding.
